# Project Documentation

This folder contains the project documentation files for the Student Result Analysis and Management System.

## Files to Add

Please add the following files to this directory:

1. **Project_Report.pdf** - Your complete project report in PDF format
2. **Presentation.pptx** - Your project presentation in PowerPoint format

## How to Upload

### Option 1: Using GitHub Web Interface
1. Navigate to this `docs/` folder on GitHub
2. Click "Add file" > "Upload files"
3. Drag and drop your files or click to browse
4. Add a commit message (e.g., "Add project report and presentation")
5. Click "Commit changes"

### Option 2: Using Git Command Line
1. Copy your files to this `docs/` folder
2. Run the following commands:
   ```bash
   git add docs/
   git commit -m "Add project report and presentation"
   git push origin main
   ```

## Supported File Formats

- **Report**: PDF (.pdf)
- **Presentation**: PowerPoint (.pptx, .ppt) or PDF (.pdf)
